# Iris Flower Classification

## Overview

This project uses the K-Nearest Neighbors (KNN) algorithm to classify iris flowers into three species: Setosa, Versicolor, and Virginica.

## Dataset

The Iris dataset contains 150 samples with the following features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies Used

* Python
* Scikit-learn
* NumPy
* Pandas

## Algorithm

K-Nearest Neighbors (KNN)

* Accuracy: 100%
* Precision: 100%
* Recall: 100%
* F1-Score: 100%
